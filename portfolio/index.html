Portfolio

Ceci décrit les projets réaliser par M2i3 pour l'un de nos clients ou nos projets internes.

  • M2i3

    Refonte de notre nouveau site web. On utilise ici le CMS Silverstripe ainsi que JQuery pour la présentation. Un petit agrégateur de nouvelle SimplePie aide à l'ajout de contenus sur une base régulière.

    • Système de gestion de contenus: Silverstripe avec JQuery et SimplePie
    • Langage de programmation: PHP
    • Base de données: MySQL

  • Artyan

    Artyan est une petite école de airbrush sur la rive sud de Montréal. Ce projet utilise les technologies de JQuery pour présenter une jolie galerie de photos et l'utilisation de Paypal pour la gestion des cours et de leurs paiements en ligne.

    • Système de gestion de contenus: Silverstripe avec JQuery, Paypal
    • Langage de programmation: PHP
    • Base de données: MySQL

  • Karaté Sunfuki

    Une des grandes école de Karaté de la région de Montréal et des environs. Ce projet met en valeur les capacités de Silverstripe à gérer plusieurs gabarits. Nous utilisons aussi JQuery pour mette le menu en valeur.

    • Système de gestion de contenus: Silverstripe avec JQuery
    • Langage de programmation: PHP
    • Base de données: MySQL

  • L'Esprit du Thé

    Ce sympathique salon de thé de la rue Laurier à Montréal nous a contacté pour réaligner le site et ajouter une plateforme de commerce électronique. Une fois la plateforme Joomla mise à jour, VirtueMart a été ajouté. Pour ce site nous avons utiliser une planoplie de composante Joomla pour assurer le bon fonctionnement et une rentabilité maximum.

    • Système de gestion de contenus: Joomla 1.5.x avec VirtueMart (Canadien), Joomfish, sh404SEF, JoomlaPack et extPlorer.
    • Langage de programmation: PHP
    • Base de données: MySQL

  • Retrouvailles de la promotion de 1994 de l'ESFL

    Qui a dit qu'on ne pouvait pas s'occuper de son Alma Mater. Ce site est une petite réalisation de Jean-Marc pour les retrouvailles de son école secondaire. Conçu en Ruby on Rails le site a pris au total 3 jours à designer visuellement et mettre en place.

    • Système de gestion de contenus: Zliki
    • Langage de programmation: Ruby on Rails
    • Base de données: MySQL

  • G-Force

    G-Force sont les spécialistes de la Diversité en emploi. Leur portail est réalisé à l'aide de Joomla et inclus les module Joomfish, ArtBanner Plus, Artio SEF, AdSense Joomlaspan et autres modules développé par M2i3 pour leur besoins.

    • Système de gestion de contenus: Joomla 1.0.x avec Joomfish, ArtBanner, Artio SEF, Joomlaspan
    • Langage de programmation: PHP/Coldfusion
    • Base de données: MySQL

    • Base de données: MySQL
